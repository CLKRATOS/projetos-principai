windows 11 ganha abas no explorador de arquivose mais novidades

Um dos recursos mais esperador no windows 11, as guias no explorador de arquivos acabam de chegar a versão de
teste do sistema operacional. elas estão entre as novidades da build 25136 liberada hoje (10) pela microsoft,
que já esta disponivel para os membros de peograma insider incritos no canal DEV.

Principal mudança adicional no reformulado da nova geração do sistema operacional, a chegada das abas
altera a dinâmica de uso da ferramenta, especialmente para quem mantém diversas páginas diferetes abertas ao
mesmo tempo. Agora, fica msis fácil gerenciar toda elas, assim como você faz em um navegador com varias sites
abertos.

também serão visualizados, novo layout do canto esquedor, as pastas do Onedrive associadas á conta uasada no
dispositivos. Ao navegar para elas, a barra de endereço exibirá o caminho correto, ajudando a esclarecer quanso
você está utilizador os arquivos salvos na nuvem ou localmente, no disco rigido do pc, conforme a microsoft.

Widgets dinâmicas e outros destaques

Esta atualização do windows 11 conta ainda com outras novidades, como a exibição de windgets dinâmics na barra
dos widgets de finanças e esportes, bem como as notificações das útimas noticias

Sengundo a big tech, isso ajudará o usuários a se manter informado, bastando clicar no quadro de widgets para
visualizar a sinformações completas. caso não haja nenhuma interação, abrra de tarefas continuará exibindo
as informações meteorológica.
